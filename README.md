# SmartCity Connect – SDG 11 Prototype

## SDG Goal
SDG 11 – Sustainable Cities and Communities

## Problem Statement
Urban cities face challenges such as inefficient waste management, delayed issue resolution,
and lack of citizen participation in city development.

## System Objective
SmartCity Connect is a digital platform that allows citizens to report urban issues
and track their resolution, promoting sustainable city development.

## Features
- Issue reporting (garbage, potholes, streetlight, water leakage)
- Image upload support
- Complaint status tracking
- Simple and user-friendly interface

## Prototype Tool
HTML Mock Page

## Agile Methodology
This prototype was developed using Agile principles with sprint-based development
and user stories.
